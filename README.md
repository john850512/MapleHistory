### 2018.11.18 - 第三章節完成。
  - 字體、介面優化，更加舒適的瀏覽體驗
  - 請下載[Release_APP/MapleHistory_1118.apk](https://github.com/john850512/MapleHistory/blob/master/Release_APP/MapleHistory_1118.apk)

<img src="img/Boss_1.png" width="200" height="200" align = "left">
第三章的內容是關於主角與女主角在天空之城組隊任務"女神之塔"發生的的故事。
<br>
在很久很久以前的楓之谷，曾經有一個bug可以快速刷女神之塔: 透過偷碎片(透過拔網路線、防火牆等技巧)的方式來快速刷組隊任務(這個偷碎片的技巧在當時不只是組隊任務，還有其他活動地圖也都可以用的到，像是~~偷叉子~~)
<br>
整段遊戲的製作是希望讓家可以從中體驗到楓之谷最原始的感動，音效、圖片上的取材花費了很多時間與心力，如果喜歡請給星星支持本專案，謝謝。
<br><br>

### 2018.9.17 - 第二章節完成。
  - 第二章節的故事背景與第一章節沒有差異太大，所以不用耗費太多心力在找尋素材上
  - 優化章節選單，現在選擇不同選單可以同時出現章節名稱
  - 我把部分字體改成開源字體了，因為前陣子字體事件~~我怕被吉~~
  - 這次暫時先開放.apk檔，請下載[MapleHistory_0917.apk](https://github.com/john850512/MapleHistory/blob/master/Release_APP/MapleHistory_0917.apk)並在手機上安裝試玩
  <img src="img/4.JPG" width="50%" >
  
<img src="img/Boss_2.png" width="200" height="200" align = "left">
第一、二章主線圍繞在主角與女主角的第一次相遇，也就是在玩具城101組隊任務的情景。
<br>
說起玩具城的組隊任務，大家一定不會忘記這個口號"133 221 333 123 111"，玩具城組隊任務在早期陪伴著許多玩家，一起打怪、一起聊天，慢慢的彼此的友誼越來越深厚......
<br>
希望大家能透過這個遊戲回憶起當初和朋友一起組隊遊玩解任務的感覺。
